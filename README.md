# GateCraft Hub

Jednoduchá demo webová aplikace pro správu pracovních hodin, skladových zásob a projektů pro firmu vyrábějící vrata, brány a ploty.

## Jak spustit

1. Otevřete [index.html](index.html) v prohlížeči.
2. Pro plnou Firebase integraci je nutné nahradit konfiguraci v souboru [index.html](index.html) vlastním Firebase projektem.
3. V Firebase Console vytvořte projekt, zapněte Authentication a Firestore a do konfigurace vložte vlastní `apiKey`, `authDomain`, `projectId`, `storageBucket`, `messagingSenderId` a `appId`.

## Cíl

- Dark UI pro management aplikaci
- Přihlášení/registrace a role admin/worker
- Základní workflow pro schvalování hodin, sklad a projekty
